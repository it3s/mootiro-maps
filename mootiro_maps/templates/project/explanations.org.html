Criação de projeto

Olá, {{ name }}!

Você acabou de criar um projeto de mapeamento no MootiroMaps.

Veja algumas dicas sobre como prosseguir:

  • Crie uma descrição para seu projeto

    Ao iniciar uma ação de mapeamento e criar um projeto é importante você identificar:
    • O que está mapeando
    • Quais os objetivos e finalidades
    • Quem são os colaboradores
    • Quem são parceiros do mapeamento - você pode carregar os logos de seus apoiadores

  • Adicione conteúdos já existente no MootiroMaps

    Agora você pode adicionar ao seu projeto conteúdos já existentes na plataforma. Siga os seguintes passos:
    1. Na página do objeto procure o botão
    2. Ao clicar, digite o nome do seu projeto;
    3. salve a alteração.

  • Importe dados para seu projeto

    Você também pode importar de uma só vez um grande volume de conteúdos. Se você já possui dados para serem importados envie um email para contato@it3s.org.


  • Divulgue seu projeto nas redes sociais

    Utilize os botões de redes sociais (Facebook, Twitter e Google+) para compartilhar e dar visibilidade aos conteúdos, além de mobilizar novos colaboradores.
